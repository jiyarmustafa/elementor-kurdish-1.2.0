بستەی تەواوی وەرگێڕانی ئێلێمنتۆر بۆ زاراوەی کوردی سۆرانی، لەگەڵ فۆنتی کوردی، پشتیوانی RTL، و تمپلیتە کوردییەکان.

== Description ==

ئەفزونەی **ئێلێمنتۆری کوردی** ڕاهێنانێکی تەواوە بۆ وێبسایتە کوردییەکان کە بە زاراوەی **سورانی** نووسراون. ئەم ئەفزونە وەرگێڕانی تەواوی ئێلێمنتۆر و ئێلێمنتۆر پرۆ دابین دەکات، فۆنتی کوردی (وەک Soran) زیاد دەکات، و تجربەی دروستکردنی وێبسایت بۆ بەکارهێنەرانی کوردی بە شێوەیەکی ڕادیکال گۆڕدەدات.

**وەرگێڕانی کوردی سورانی**  
پاش فەعّالکردنی ئەفزونەکە، ڕووکاری بەکارهێنەری ئێلێمنتۆر و ئێلێمنتۆر پرۆ بە شێوەیەکی خۆکارانە بە **کوردی سورانی** دەربکەوێت. ئەم وەرگێڕانە پێویستی بە هیچ ڕێکخستنێک نییە و بەکارهێنانی ئێلێمنتۆر بۆ بەکارهێنەرانی کوردی سادەتر دەکات.

**فۆنتی کوردی**  
وەرگێڕانی کوردی سورانی، فۆنتی کوردی (Unikurd Goran, Kurdistan 24, Rudaw, Shasenem)، و تایبەتمەندییە منحەصەربەفەرد.

**پشتیوانی RTL**  
هەموو ڕووکاری ئێلێمنتۆر (ویرایشگەر، پێشبینین، داشبۆرد) بۆ **ڕاست بۆ چەپ** (RTL) گونجاوە، بە تایبەتی بۆ زمانی کوردی.

**تمپلیتە کوردییەکان**  
بەستەرێک بۆ **تمپلیتە کوردییەکان** لە کتێبخانەی ئێلێمنتۆر زیاد کراوە، کە لە سایتی [jiyarwp.ir/templates](https://jiyarwp.ir/templates) دەردەکەوێت.

**دەستپێکی کوردی**  
لاپەڕەیەکی تایبەت لە داشبۆردی وردپرس بۆ ڕاهێنانی بەکارهێنەری کوردی دروست کراوە.

== Installation ==

1. پوشه‌ی `elementor-kurdish` بخە بۆ `/wp-content/plugins/`
2. ئەفزونەکە لە ڕێگەی مێنۆی 'ئەفزونەکان' لە وردپرس فەعّال بکە
